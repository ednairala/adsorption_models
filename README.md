# adsorption_models

Adsorption Isotherm Models - Object-Oriented Version
----------------------------------------------------
This class-based Python implementation contains multiple adsorption
isotherm models (Henry, Langmuir, BET, Freundlich, Temkin,
Dubinin–Radushkevich, and Redlich–Peterson).

It can:
 - Access individual models as methods.
 - Plot them all for comparison.
 - Later: fit experimental data to them (optional extension).
